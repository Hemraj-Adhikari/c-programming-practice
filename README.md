#  C Programming Practice (Basic)

##  Introduction
This repository contains **basic C programming programs** designed for **CSIT / BIT students** and beginners.

It focuses on fundamental programming concepts, logical thinking, and practical coding skills required for academic learning.

---

## Objective
- Build a strong foundation in C programming  
- Improve problem-solving and logical thinking  
- Support academic assignments and practical exams  

---

##  Topics Covered

###  Basic Concepts
- Hello World Program  
- Input / Output (`printf`, `scanf`)  
- Data Types & Variables  
- Operators  

###  Control Statements
- `if`, `if-else`, `nested if`  
- `switch`  
- Loops (`for`, `while`, `do-while`)  

###  Functions
- Function declaration & definition  
- Parameter passing  
- Return values  
- Basic recursion  

###  Arrays
- One-dimensional arrays  
- Array input/output  
- Sum, average, maximum, minimum
